# 🚀 Fake News Detection using NLP

## 🔗 Live Demo  
👉 https://news-fake-checker.streamlit.app/

---

## 📌 Overview  
This project detects whether a news article is Fake or Real using Machine Learning and Natural Language Processing.

---

## ⚙️ Technologies Used  
- Python  
- TF-IDF Vectorizer  
- Logistic Regression  
- Streamlit  

---

## 🧠 How it Works  
Text input is converted into numerical features using TF-IDF and classified using a Logistic Regression model.

---

## 📊 Results  
- Achieved ~98% accuracy  
- Evaluated using Confusion Matrix
